## How to use:
1. Clone the latest release 
```
git clone https://github.com/godwers/Beecrowd-downloader.git
```
2. Create a virtual enviroment in Python 
```
python3 -m venv venv
```
3. Activate the virtual enviroment
> Now it depends of the OS that you are using to activate it.<br> Please look into the [Python virtual enviroment docs](https://docs.python.org/3/library/venv.html#how-venvs-work)
```
Example on Linux:
$ source venv/bin/activate
```
4. Download python requirements using pip
```
pip install -r requirements.txt
```
5. Now you can run the code!!!!11111
```
python main.py
```
